 Möbius Spatetime Theory (MST) Visualizer
**Researcher:** Chris (Nairobi, Kenya)

## Abstract
This project is a computational exploration of the **Möbius Spatetime Theory (MST)**. MST posits that the global structure of time is a non-orientable manifold. This visualizer allows users to interact with a 3D model of this structure, simulating the "topological flip" and the influence of entropy over a 4π temporal rotation.

## Core Features
- **3D Manifold Rendering:** Visualizing the Möbius topology using parametric equations.
- **Entropy Simulation:** A stochastic noise slider to model thermodynamic decay.
- **Research Sidebar:** Integration of "Page 7" notes, including assumptions, limitations, and open questions.

## How to Run
This application is built with **Streamlit**.
